# To-Do-List
To-Do List is a simple web application to save your daily tasks in order not to miss anything.

## Features
- Add New Task
- Mark task as completed/uncompleted
- Set task priority
- Delete single task
- Delete Completed Task

## Technologies Used
- HTML
- CSS3
- Javascript
